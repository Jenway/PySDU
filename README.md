# SDU SSO login

Current features:

- login to SDU SSO
- get user info
- get lessons from bkzhjx.wh.sdu.edu.cn

## Install

```bash
pdm install
```

or 

```bash
pip install -r requirements.txt
```

## Usage

```bash
python ./src/main.py  # get user info
python ./src/bkzhjx_login.py  # get lessons
```
